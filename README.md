# Agent-example

**实现简单的上下文管理、记忆摘要、function calling**

1. 安装 uv https://docs.astral.sh/uv/getting-started/installation/

2. 修改.env

3. ```
   uv run agent.py .
   ```

> [!TIP]
>
> 项目上下文 │ <project_dir>/.agent/memory.json
>
> 用户偏好   │ ~/.agent/preferences.json
>
> 对话历史│<project_dir>/.agent/history.json（最近10条）
